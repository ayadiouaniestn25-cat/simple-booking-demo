# simple-booking-demo
Product Name:
Simple Appointment Booking (LocalStorage)

Description:
Simple Appointment Booking is a lightweight and easy-to-use booking script built with pure HTML and JavaScript.
It allows users to book appointments without the need for a backend or database.
Ideal for small businesses, freelancers, salons, and personal projects.

Features:
- Simple appointment booking form
- Stores bookings using browser LocalStorage
- No database or backend required
- Works offline
- Fully responsive design
- Easy to customize and integrate

Requirements:
- Modern web browser (Chrome, Firefox, Edge, Safari)
- No server or hosting required

Installation:
1. Download and extract the files
2. Open the project folder
3. Open index.html in your browser
4. The booking system is ready to use

Customization:
- Edit texts directly in index.html
- Change colors and styles inside the <style> section
- Modify booking logic in app.js

Data Storage Notice:
This script uses browser LocalStorage to store booking data.
Clearing browser data will remove all saved appointments.
